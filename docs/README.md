# Obsidian Core Implementation Guides

This directory contains guides for implementing Obsidian Core in different programming languages and frameworks.

## Available Guides

- [Build Guide](./build-guide.md) - Platform-specific build instructions
- [Token Guide](./token-guide.md) - Custom token creation and management
- [Go Implementation](./go-implementation.md) - Reference implementation
- [Rust Implementation](./rust-implementation.md) - High-performance systems
- [Python Implementation](./python-implementation.md) - Research and scripting
- [JavaScript/Node.js Implementation](./javascript-implementation.md) - Web integration
- [Protocol Specification](./protocol-spec.md) - Wire protocol details

## Getting Started

Choose an implementation guide based on your use case:

- **Production Node**: Use Go or Rust implementation
- **Research/Experimentation**: Use Python implementation
- **Web Applications**: Use JavaScript implementation
- **Custom Integration**: Follow the protocol specification

## Contributing

To add a new implementation:

1. Create a new markdown file in this directory
2. Follow the structure of existing guides
3. Include code examples and setup instructions
4. Test your implementation against the reference Go client

## Protocol Compatibility

All implementations must be compatible with the Obsidian wire protocol. See [protocol-spec.md](./protocol-spec.md) for details.</content>
